CraftIq Web Development

Developed CraftIQ, a full-stack skill-sharing and learning platform using Spring Boot and React with a MySQL database. The platform fosters a collaborative community where users can share skills, engage with others, and track their learning progress visually.

Key features include Skill Post Management, Learning Plans, and User Interaction through likes, comments, and feedback. The platform also offers a Progress Dashboard that displays learning statistics and engagement metrics using interactive charts.

Technologies: Spring Boot, React, Axios, SweetAlert2
Database: MySQL
